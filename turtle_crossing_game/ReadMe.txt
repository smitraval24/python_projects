This is a turtle crossing a road full of cars/moving objects.
By each increase in the level(succesfull road crossing) the speed of the cars increases.
Use upper key to move forward. Down arrow key does not work to maintain some spice in the game.
Programming language used here is Python.
OOP concept is been used here.
To execute the code run the main.py file.
Turtle class used to make game articles and the screen.
